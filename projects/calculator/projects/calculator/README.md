# Продвинутый Калькулятор

Мой первый серьёзный мини-проект на Python.

### Возможности:
- 6 математических операций (+, -, *, /, %, **)
- Обработка ошибок (`try-except`)
- Защита от деления на ноль
- Цикл — можно считать сколько угодно примеров
- Красивый интерфейс

### Чему я научился:
- Работать с `while True`
- Использовать `try-except` для обработки ошибок
- Работать с `float` и форматированными строками (f-strings)
- Структурировать код

Это один из первых проектов, где я начал переходить от простых скриптов к более продуманным программам.
